# Project 2 - Image Stitching

## Operating System

    macOS Big Sur 11.6

## Language

    Python 3.9.7

## Dependency

    numpy==1.21.4
    opencv-python==4.5.4.60
    matplotlib==3.2.2

或是可以在```code```資料夾底下執行```pip3 install -r requirements.txt```。

## Usage

在```code```資料夾底下執行：

    python3 main.py

或是可以使用```jupyter notebook```，執行```code```資料夾底下的```main.ipynb```。
